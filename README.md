# Discord-Bot-Codes

Python (.py) files containing public Discord bot templates and utilities.

## Requirements for diagnosticbot.py

- Python: 3.10 <= version < 3.13
- discord.py (v2.5.1)
- paramiko (for SSH support)
- aiohttp (for async HTTP requests)
- psutil (for system statistics: CPU, RAM, Disk, uptime)

Recommended install (pip):
```bash
python -m pip install "discord.py==2.5.1" "paramiko==3.4.3" "aiohttp==3.9.4" "psutil==5.9.7"
```

Notes
- Ensure you run the bot with a compatible Python interpreter.
- Adjust dependency versions if necessary for your environment.
